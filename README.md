# 📊 Data Analyzer and Transformer (Python)

## 🚀 Overview
A menu-driven Python application that demonstrates:
- 📥 Input Data
- 📈 Data Summary (built-in functions)
- ♻️ Recursive Factorial
- 🎯 Lambda-based Filtering
- 🔃 Sorting (Ascending/Descending)
- 📊 Dataset Statistics (multiple return values)
- 🧩 `**kwargs` summary
- ❌ Exit Menu

## 🛠️ Features
1. Input custom or sample data
2. Display min, max, sum and average
3. Calculate factorial using recursion
4. Filter values using a lambda function
5. Sort data
6. Display dataset statistics
7. View tracked metrics
